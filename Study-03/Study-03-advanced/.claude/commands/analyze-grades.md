# analyze-grades

전체 학생 성적을 분석하는 명령어

## 실행
```bash
node grade-analyzer.js
```

## 기능
- 전체 평균 점수 계산
- 등급별 분포 시각화
- 반별 평균 성적 비교
- 카테고리별 평균 정답률
- 상위/하위 5명 추출
- 최근 성적 추이 분석

## 전제조건
- `student_data.json` 파일 필요

## 출력 예시
```
📊 성적 분석 보고서
========================
📈 전체 통계
  • 총 학생 수: 20명
  • 전체 평균 점수: 67.9점

📊 등급별 분포
  A: ██ 1명 (5.0%)
  B: ████ 2명 (10.0%)
```
