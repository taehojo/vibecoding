# compare-students

학생 간 성적을 비교 분석하는 명령어

## 실행
```bash
# 기본 실행 (상위 3명 vs 하위 3명)
node student-comparison.js

# 특정 학생 지정
node student-comparison.js STU001 STU005 STU010
```

## 기능
- 기본 정보 테이블 (평균, 등급, 최고/최저 점수)
- 카테고리별 정답률 비교
- 성적 추이 분석 (상승/하락 표시)
- 평균 학습 시간 비교
- 개별 강점/약점 분석
- 그룹 통계

## 출력 형식
- 테이블 형식의 비교 데이터
- 시각적 막대 그래프
- 추세 아이콘 (↑/↓/→)
