# 책 이후의 업데이트 - marketplace로 PRD 만들기

## 개요

Claude Marketplace에서 유용한 프롬프트를 검색하고, 이를 SKILL.md로 저장하여 클로드 코드에서 재사용하는 방법을 배웁니다. 예시로 PRD(제품 요구사항 문서) 생성 프롬프트를 활용합니다.

## Step 1: Claude Marketplace 접속

구글에서 "claude marketplace"를 검색하거나, 아래 주소로 직접 접속합니다:

👉 **https://claudemarketplaces.com/**

Claude Marketplace는 다양한 사용자들이 만든 프롬프트와 플러그인을 공유하는 커뮤니티입니다.

## Step 2: 플러그인 설치

클로드 코드 터미널에서 다음 명령어를 입력합니다:

```
/plugin marketplace add f/awesome-chatgpt-prompts
```

설치가 완료되면 다양한 역할 기반 프롬프트에 접근할 수 있게 됩니다.

## Step 3: 프롬프트 검색

클로드 코드에 다음과 같이 요청합니다:

```
PRD 생성 프롬프트를 검색해줘
```

클로드 코드가 marketplace에서 PRD(Product Requirements Document) 관련 프롬프트를 검색하여 보여줍니다.

## Step 4: 검색된 프롬프트 확인

검색 결과에서 PRD 생성에 적합한 프롬프트를 확인합니다. 예를 들어:

```
당신은 시니어 프로덕트 매니저입니다. 다음 제품에 대한 PRD를 작성해주세요:
- 제품명: {{product_name}}
- 목표 사용자: {{target_user}}
- 핵심 기능: {{core_features}}

PRD에는 다음 섹션을 포함해주세요:
1. 제품 개요
2. 목표 및 성공 지표
3. 사용자 스토리
4. 기능 요구사항
5. 비기능 요구사항
6. 타임라인
```

## Step 5: SKILL.md로 저장하여 재사용

검색된 프롬프트를 SKILL.md 형태로 저장하면, 프로젝트마다 재사용할 수 있습니다.

클로드 코드에 다음과 같이 요청합니다:

```
이 PRD 프롬프트를 .claude/skills/prd-generator/SKILL.md로 저장해줘
```

생성되는 SKILL.md 예시:

```markdown
# PRD Generator

## 설명
제품 요구사항 문서(PRD)를 체계적으로 생성하는 스킬입니다.

## 사용법
"PRD를 작성해줘"라고 요청하면 이 스킬이 활성화됩니다.

## 프롬프트
당신은 시니어 프로덕트 매니저입니다. 사용자가 제공하는 정보를 바탕으로
체계적인 PRD를 작성합니다.

### 필수 입력 정보
- 제품명
- 목표 사용자
- 핵심 기능

### 출력 형식
1. 제품 개요
2. 목표 및 성공 지표 (KPI 포함)
3. 사용자 스토리 (As a... I want... So that...)
4. 기능 요구사항 (우선순위별)
5. 비기능 요구사항
6. 기술 제약사항
7. 타임라인 및 마일스톤
```

## Step 6: 저장된 스킬 활용

이제 어떤 프로젝트에서든 클로드 코드에게 "PRD를 작성해줘"라고 요청하면, 저장된 SKILL.md의 형식에 따라 체계적인 PRD가 생성됩니다.

```
새로운 프로젝트의 PRD를 작성해줘.
제품명: AI 학습 도우미
목표 사용자: 대학생
핵심 기능: 강의 요약, 퀴즈 생성, 학습 계획 수립
```

## 핵심 포인트

- **Marketplace**: 다른 사용자들이 검증한 고품질 프롬프트를 바로 활용할 수 있습니다
- **SKILL.md**: 프롬프트를 스킬로 저장하면, 매번 처음부터 프롬프트를 작성할 필요 없이 일관된 결과를 얻을 수 있습니다
- **재사용성**: 한번 저장한 스킬은 모든 프로젝트에서 활용 가능합니다
